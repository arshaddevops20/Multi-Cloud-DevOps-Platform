# Azure Kubernetes Service (AKS)

## Features

- Azure CNI Overlay
- User Assigned Managed Identity
- Azure RBAC
- OIDC Issuer
- Workload Identity
- Autoscaling
- Multiple Node Pools
- Log Analytics Integration
- Microsoft Defender Ready
- ACR Integration

## Node Pools

- System
- Apps

## Outputs

- Cluster ID
- Cluster Name
- OIDC Issuer URL
- Node Resource Group
- FQDN
